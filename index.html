GraphKit

The Modern C++ Graph Library

v1.0.0 C++17 Header-Only MIT License
Get Started View on GitHub

GraphKit is a high-performance, header-only C++17 library designed for graph algorithms and visualization. Built for students, researchers, and developers, it provides a simple yet powerful API for working with graphs.

Why GraphKit?

Lightning Fast

Optimized algorithms with modern C++17 features for maximum performance in graph computations.

Zero Dependencies

Header-only library with no external dependencies. Just include and start coding immediately.

DOT Visualization

Generate DOT format code for creating beautiful graph diagrams with online Graphviz tools.

Generic Design

Template-based architecture supporting any data type as graph nodes with full type safety.

Quick Start

main.cpp 
#include "graphkit.hpp"
#include <iostream>

int main() {
    // Create a directed graph
    gk::Graph<std::string> g(gk::GraphType::DIRECTED);
    
    // Add edges with weights
    g.addEdge("A", "B", 4);
    g.addEdge("A", "C", 2);
    g.addEdge("B", "D", 5);
    g.addEdge("C", "D", 1);
    g.addEdge("D", "E", 3);
    
    // Find shortest path
    long long distance = g.getShortestPath("A", "E");
    std::cout << "Shortest path from A to E: " << distance << std::endl;
    
    // Perform BFS traversal
    auto result = g.bfs("A");
    std::cout << "BFS order: ";
    for (const auto& node : result.order) {
        std::cout << node << " ";
    }
    
    return 0;
}

Installation

Step 1: Download the Headers

GraphKit consists of two header files:

  • graphkit.hpp - Core graph algorithms and data structures
  • graph_viz.hpp - Visualization functions (optional, for DOT generation)
# Download both files
wget https://raw.githubusercontent.com/lakshhttps/GraphKit/main/graphkit.hpp
wget https://raw.githubusercontent.com/lakshhttps/GraphKit/main/graph_viz.hpp

# Or clone the entire repository
git clone https://github.com/lakshhttps/GraphKit.git

Step 2: Include in Your Project

Add the headers to your C++ source file and compile with C++17 or later.

#include "graphkit.hpp"       // Core library
#include "graph_viz.hpp"      // For visualization (optional)

// Start using GraphKit!
gk::Graph<int> g;

Compile your program:

g++ -std=c++17 main.cpp -o program

# Or with clang
clang++ -std=c++17 main.cpp -o program

API Reference

Namespace

All GraphKit classes and functions are within the gk namespace:

using namespace gk;  // Option 1: Use entire namespace

gk::Graph<int> g;    // Option 2: Prefix with gk::

Graph Construction

Constructor

Graph<T>(GraphType gType = GraphType::DIRECTED)

Description: Creates a new graph with the specified type.

Parameters:

  • gType - Graph type: GraphType::DIRECTED or GraphType::UNDIRECTED (default: DIRECTED)
// Directed graph
gk::Graph<int> directedGraph(gk::GraphType::DIRECTED);

// Undirected graph
gk::Graph<std::string> undirectedGraph(gk::GraphType::UNDIRECTED);

addEdge

void addEdge(T u, T v, int w = 1)

Description: Adds an edge from node u to node v with weight w. For undirected graphs, automatically adds the reverse edge.

Parameters:

  • u - Source node
  • v - Destination node
  • w - Edge weight (default: 1)
g.addEdge("A", "B", 5);  // Add edge A->B with weight 5
g.addEdge("B", "C");     // Add edge B->C with default weight 1

size

std::size_t size() const

Description: Returns the total number of nodes in the graph.

std::size_t numNodes = g.size();
std::cout << "Graph has " << numNodes << " nodes" << std::endl;

Traversal Algorithms

bfs

TraversalResult<T> bfs(T start) const

Description: Performs Breadth-First Search starting from the given node.

auto result = g.bfs("A");
for (const auto& node : result.order) {
    std::cout << node << " ";
}

dfs

TraversalResult<T> dfs(T start) const

Description: Performs Depth-First Search starting from the given node.

auto result = g.dfs("A");
std::cout << "DFS traversal: ";
for (const auto& node : result.order) {
    std::cout << node << " ";
}

topologicalSort

std::vector<T> topologicalSort() const

Description: Returns a topological ordering of vertices in a directed acyclic graph (DAG).

// Course prerequisites example
g.addEdge("Math101", "Math201");
g.addEdge("Math201", "Math301");
auto order = g.topologicalSort();
// Output: Math101, Math201, Math301

Shortest Path Algorithms

getShortestPath

long long getShortestPath(T start, T end) const

Description: Finds the shortest path distance between two nodes.

long long dist = g.getShortestPath("A", "E");
if (dist == -1) {
    std::cout << "No path exists" << std::endl;
} else {
    std::cout << "Shortest distance: " << dist << std::endl;
}

getAllPairsShortestPaths

std::map<T, std::map<T, long long>> getAllPairsShortestPaths() const

Description: Computes shortest paths between all pairs of nodes using Floyd-Warshall.

auto distances = g.getAllPairsShortestPaths();
std::cout << "Distance from A to D: " << distances["A"]["D"] << std::endl;

getMinimumSpanningTree

MSTResult<T> getMinimumSpanningTree() const

Description: Finds the Minimum Spanning Tree using Prim's algorithm.

auto mst = g.getMinimumSpanningTree();
std::cout << "MST total cost: " << mst.totalCost << std::endl;
for (const auto& edge : mst.edges) {
    std::cout << edge.first << " - " << edge.second << std::endl;
}

Graph Properties

hasCycle

bool hasCycle() const
if (g.hasCycle()) {
    std::cout << "Graph contains a cycle" << std::endl;
} else {
    std::cout << "Graph is acyclic" << std::endl;
}

countComponents

std::size_t countComponents() const
std::size_t components = g.countComponents();
std::cout << "Graph has " << components << " component(s)" << std::endl;

getNeighbors

std::vector<T> getNeighbors(T u) const
auto neighbors = g.getNeighbors("A");
std::cout << "Neighbors of A: ";
for (const auto& node : neighbors) {
    std::cout << node << " ";
}

indegree

int indegree(T u) const
int inDeg = g.indegree("B");
std::cout << "In-degree of B: " << inDeg << std::endl;

outdegree

int outdegree(T u) const
int outDeg = g.outdegree("B");
std::cout << "Out-degree of B: " << outDeg << std::endl;

Visualization Functions

GraphKit includes visualization support through DOT format generation. Include graph_viz.hpp to use these functions.

printGraph

void printGraph(const Graph<T>& g)

Description: Generates and prints DOT code for visualizing your graph.

#include "graph_viz.hpp"

gk::Graph<std::string> g(gk::GraphType::DIRECTED);
g.addEdge("A", "B", 5);
g.addEdge("B", "C", 3);

gk::printGraph(g);  // Prints DOT code to console

printMST

void printMST(const MSTResult<T>& mstResult)

Description: Generates and prints DOT code for visualizing a Minimum Spanning Tree.

#include "graph_viz.hpp"

auto mst = g.getMinimumSpanningTree();
gk::printMST(mst);  // Prints MST DOT code with total cost

Examples

Finding Shortest Path

gk::Graph<std::string> g(gk::GraphType::DIRECTED);

g.addEdge("NYC", "LA", 2800);
g.addEdge("NYC", "Chicago", 790);
g.addEdge("Chicago", "LA", 2015);

long long dist = g.getShortestPath("NYC", "LA");
std::cout << "Distance: " << dist << " miles" << std::endl;

Detecting Cycles

gk::Graph<int> g(gk::GraphType::DIRECTED);

g.addEdge(1, 2);
g.addEdge(2, 3);
g.addEdge(3, 1); // Creates a cycle

if (g.hasCycle()) {
    std::cout << "Graph contains a cycle!" << std::endl;
}

Graph Visualization

#include "graphkit.hpp"
#include "graph_viz.hpp"

gk::Graph<std::string> g(gk::GraphType::UNDIRECTED);
g.addEdge("A", "B", 2);
g.addEdge("B", "C", 3);
g.addEdge("C", "D", 1);

// Print DOT code for visualization
gk::printGraph(g);

// Visualize MST
auto mst = g.getMinimumSpanningTree();
gk::printMST(mst);
